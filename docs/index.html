Rodando Localmente o Git Pages

Requisitos

  • Python
  • pip
  • mkdocs

Uso

Para rodar a documentação localmente, siga estes passos:

  1. Instale o tema material do mkdocs:
    2. pip install mkdocs-material
  2. Execute o comando para iniciar o servidor local:
    3. mkdocs serve

Acesse a documentação local em: http://127.0.0.1:8000/ ou http://localhost:8000/.
